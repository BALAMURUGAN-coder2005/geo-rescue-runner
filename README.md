
## 🎮 Geo Rescue Runner

A fast-paced arcade-style rescue game where players navigate through dangerous environments, avoid obstacles, and complete rescue missions with strategic movement and quick decision-making.

🔗 Live Demo: https://geo-rescue-runner.vercel.app/

### Features
- Interactive gameplay experience
- Rescue mission challenges
- Smooth animations and controls
- Responsive modern design
- Browser-based game deployment

### Technologies Used
- HTML
- CSS
- JavaScript
- Vercel
